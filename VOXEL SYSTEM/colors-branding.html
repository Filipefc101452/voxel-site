Como Alterar as Cores

✨ Para mudar as cores globalmente, edite apenas o arquivo styles.css

Passo 1: Localize as variáveis CSS

:root { --primary-color: #6366f1; --secondary-color: #ec4899; --dark-color: #1f2937; --light-color: #f9fafb; --text-color: #374151; --border-color: #e5e7eb; }

Passo 2: Substitua pelos códigos HEX desejados

:root { --primary-color: #seu_novo_roxo; --secondary-color: #sua_nova_rosa; --dark-color: #seu_novo_preto; --light-color: #seu_novo_cinza_claro; --text-color: #sua_cor_texto; --border-color: #sua_cor_borda; }

Passo 3: Salve o arquivo

Pronto! Todas as cores do site mudarão automaticamente.
